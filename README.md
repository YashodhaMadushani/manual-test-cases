#LMS Login Page Manual Test Cases

This repository contains manual test cases for the Login page of a Learning Management System (LMS). The test cases were created as part of a software testing academic exercise to practice writing structured and comprehensive manual test cases.

The objective of this project is to verify the functionality and usability of the LMS login page by testing different user scenarios, including valid and invalid login attempts, input validation, and common login features.

Note

These test cases were created for educational purpose as part of a software testing practice activity. Any application names, URLs, credentials, or other organization-specific information have been replaced with generic examples where appropriate.
